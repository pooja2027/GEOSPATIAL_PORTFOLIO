## 📂 Projects

### 🌿 [NDVI Vegetation Monitoring](./NDVI_Analysis/)
Automated NDVI analysis for any location using Sentinel-2 imagery via Google Earth Engine.
Computes annual NDVI trends (2018–2024), seasonal variation, vegetation classification by area,
change detection, and exports an interactive map + structured report.

**Stack:** Python · GEE · Streamlit · Folium · OSMnx · GeoPandas  
**Key output:** Chennai shows −10.95% vegetation decline (2018→2024), consistent with urban expansion.

---

### 🏗️ [GeoVista — AEC Market Insights](./GeoVista/)
Interactive dashboard for the Architecture, Engineering & Construction (AEC) industry.
Upload project CSV data, apply KMeans or DBSCAN geographic clustering,
visualize on interactive maps with cost heatmap layers, and export results as GeoJSON.

**Stack:** Python · Streamlit · scikit-learn · Folium · GeoPandas  
**Origin:** Built for the Code; Without Barriers Hackathon (Year 1), rebuilt and restructured in Year 3.
